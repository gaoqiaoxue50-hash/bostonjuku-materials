# ボスキャリ対策ノウハウ教材

> ボストン就活塾（note: bostonjuku）の無料記事から抽出・再構成した学習教材。  
> 各カテゴリは **Part 1（エグゼクティブサマリー）→ Part 2（記事別詳細）** の二層構造。

---

## 読み方

1. **左のサイドバー**から気になるカテゴリを選ぶ
2. **Part 1** で全体像・重要フレームワーク・NGパターンを把握する（10〜15分）
3. **Part 2** で気になる記事を深読みする
4. 元記事リンクから原文を読んでさらに理解を深める
