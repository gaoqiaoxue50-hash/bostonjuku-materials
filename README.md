# ボスキャリ対策ノウハウ教材

> ボストン就活塾（note: bostonjuku）の無料記事から抽出・再構成した学習教材。  
> 各カテゴリは **学習マップ → Part 1（必須知識）→ Part 2（記事別詳細）** の3層構造。

---

## 🗺️ 学習パス（目的別おすすめルート）

### ルートA：IBD・外銀を本気で狙う人
```
01 基礎知識（全体像・スケジュール把握）
  ↓
02 IBD外銀（部門選び → テクニカル → 志望動機）← メイン
  ↓
07 業界分析（気になるM&Aの調べ方）
  ↓
06 志望動機（NGリスト除去 → 再構築）
```

### ルートB：総合商社を本気で狙う人
```
01 基礎知識（全体像・商社選考スケジュール）
  ↓
04 商社コンサル（面接100選 + NG志望動機）← メイン
  ↓
06 志望動機（本音ベースの組み立て方）
  ↓
08 Q&A（ガクチカ・チームvs個人の悩みを解消）
```

### ルートC：サマーインターン全落ちから逆転したい人
```
09 キャリア（逆転7ステップを把握）
  ↓
06 志望動機（志望動機の根本を作り直す）
  ↓
02 IBD外銀 または 04 商社コンサル（知識を入れ直す）
  ↓
07 業界分析（「気になるM&A/案件」を一つ深掘りする）
```

### ルートD：交換留学生・Coop・コミカレ出身で不安な人
```
01 基礎知識（参加資格・採用優先度を正確に把握）
  ↓
08 Q&A（自分と似た経歴の相談事例を探す）
  ↓
09 キャリア（交換留学生が失敗しやすい理由を確認）
  ↓
業界を決めたら → ルートA または ルートB へ
```

### ルートE：コンサルを受ける人
```
04 商社コンサル（Big4 FASの選考特性を把握）
  ↓
05 GD・ケース（ケース面接の5ステップを習得）
  ↓
06 志望動機（FAS・コンサル向けの志望動機を組み立てる）
```

## 読み方

1. まず **学習パス** から自分に合ったルートを選ぶ
2. 各ファイルの **🗺️ 学習マップ** でそのカテゴリの全体像を把握
3. **📌 Part 1** でサブテーマ別の詳細ノートを読む（ここが本体）
4. **📚 Part 2** で気になる記事の全文を確認する
