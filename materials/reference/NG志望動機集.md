# NG志望動機 完全集（業界別）

> 投資銀行IBD・アセットマネジメント・総合商社のNG志望動機を業界横断で整理。  
> 「なぜNGなのか」を理解することで、自分の志望動機のリスクを事前に排除する。

**出典**:  
- [投資銀行IBDの良くあるNG志望動機](https://note.com/bostonjuku/n/n2d5be283550c)（10いいね）  
- [アセマネの良くあるNG志望動機](https://note.com/bostonjuku/n/n8049d81b19b3)（12いいね）  
- [総合商社の良くあるNG志望動機](https://note.com/bostonjuku/n/nfc74b3265b77)（12いいね）

## 重要な前提

> 「志望動機に正解はないが、不正解は確実に存在する。」

NG志望動機でも**論理を明確に説明できれば通過できる場合がある**。  
だが、それは難しい。通過率の高い志望動機とNG志望動機では勝率に大きな差がある。

**自分の志望動機チェックリスト**（提出前に確認すること）:
- [ ] 「この会社でなくてもできることではないか？」
- [ ] 「抽象的で突っ込まれたら即死するワードが入っていないか？」
- [ ] 「面接官が聞き飽きている表現を使っていないか？」
- [ ] 「志望動機が志望職種・業務内容と論理的に繋がっているか？」

## 第1部：投資銀行IBD

### NGリスト（12項目）

| NG志望動機 | 面接官の心の声・反論 |
|---|---|
| 日本経済を成長させたい | M&Aでどうやって成長させるのか？経産省や政策金融機関でも良いのでは |
| 金融は経済の血液 | 概念的すぎる。聞き飽きた |
| 日本企業の海外プレゼンスを上げたい | 「プレゼンス」とは具体的に何を指すのか |
| 体育会で培ったリーダーシップとチームワークを活かしたい | それはどの仕事でも使える強みでは |
| 起業/ファンドへの修行として | 最初から出口を決めて入社するのか |
| 専門性が欲しい | なぜIBDの専門性でなければいけないのか |
| 英語を使う仕事がしたい | 他に英語を使う職種はいくらでもある |
| 大規模なことをやりたい | プロジェクトファイナンスや官民PPPの方が規模は大きい |
| 顧客に貢献したい | どんな顧客に？どのような形で貢献するのか |
| 社会に貢献したい | M&Aは社会貢献か？具体的な貢献の経路は |
| 金融や会計の知識を活かしたい | 激詰めに合う。知識を活かすだけなら会計士でも良いのでは |
| 企業分析がしたい | 学者になってください |

---

### IBDで「通る」志望動機の方向性

- **受験先が担当した具体的なM&A案件を挙げ、「この取引の何が戦略的に意義深いか」を語れる**
- 「企業の重要な意思決定（M&A・資金調達・上場）をアドバイザーとして支える役割に就きたい」という**役割への理解**から入る
- 自分のバックグラウンド（留学中の企業買収ニュースとの出会い・財務モデリングへの興味等）と**具体的な接点**を持つ

| 成長産業だから | その恩恵を享受したいだけのフリーライダーに見える |
| 人のお金を増やしたいから | 偽善的に聞こえる。手数料もらって自分も儲ける仕事では |
| 株が好きだから | 個人投資家になればよいのでは |
| 専門性が欲しい | なぜ「運用」の専門性でないといけないのか |
| 貯蓄から投資へ〜 | ニュースのスローガンをそのまま使わない |
| トレーダーになりたいから | アセマネのトレーダーは投資銀行のトレーダーと大きく異なる（役割が地味） |

| 投資で日本経済に貢献したいから | 「俺はなんら価値を生み出していない、ただ価値を保有しているだけだ」（映画"ウォール街"のゲッコーの言葉。アセマネの本質を理解しているか試される） |

---

### アセマネで「通る」志望動機の方向性

- **投資哲学の明確化**: 「どのような銘柄を、なぜ評価するのか」を語れる
- **CFA・証券アナリスト**など専門資格への意欲（学歴が低くても資格でカバーできる業界）
- 「株価の動きではなく、企業の本源的価値（Intrinsic Value）に関心がある」という**長期投資家の視点**
- 実際に個人で投資をしており、自分の投資判断の根拠を論理的に語れる

| NG志望動機 | 面接官の心の声・反論 |
|---|---|
| 途上国を救いたい | 途上国に行ったことはあるのか。具体的なビジネスプランは |
| 再生エネルギーをやりたい | 聞き飽きた。どのセグメントで、どの商社の強みを活かすのか |
| 風力発電をやりたい | 専業エネルギー会社でできるのでは |
| 海水淡水化をやりたい | 同上 |
| 繊維をやりたい（伊藤忠） | 伊藤忠の繊維事業の具体的な強みを説明できるか（激詰めされる） |
| エネルギーの安定供給に貢献したい | ENEOSやINPEXでも良いのでは |
| 事業投資をやりたい | 商社の仕事の大半が事業投資。差別化できていない |
| 事業経営をやりたい | 起業してください |
| グローバルに働きたい | 外資系企業・外資系コンサルでも同様のことができるのでは |
| モビリティ事業をやりたい | なぜ商社のモビリティ事業なのか。テック企業でも同様に可能では |
| ヘルスケア事業（健康寿命延伸）をやりたい | — |
| 人を巻き込んでプロジェクト管理をしたい | ガクチカを無理やり商社業務に結びつけた印象。商社マンの仕事とずれている |
| 日本のプレゼンスを上げたい | 「プレゼンス」とは？外交官でも同様にできるのでは |
| 日本と海外の懸け橋になりたい | 具体的に何をするのか。外交官は？国際機関は？ |
| AI事業をやりたい | テック専業企業（Google・Microsoft・国内IT）に行く方が深くできる |
| 社会に貢献したい | 商社が社会に貢献するメカニズムを具体的に説明できるか |

---

### 商社で「通る」志望動機の方向性

**受験先の実際の案件を使う（最重要）**:
- 三菱商事: Aethon Energy買収（1.2兆円・LNG）、メルカリ初期投資
- 三井物産: 豪州鉄鉱石権益（物産史上最大級）
- 伊藤忠: ファミリーマート完全子会社化

**論理の構造**:
```
「自分がやりたいこと（具体的な事業）」
  ↓
「なぜ商社でないとできないのか（商社のスケール・ネットワーク・資本力）」
  ↓
「なぜその商社でないとできないのか（各社の強み・文化との相性）」
  ↓
「入社後のキャリアパスと貢献方法」
```

## 第4部：業界横断 NG表現一覧

**以下の表現はどの業界でも使わない**:

| NG表現 | 理由 |
|---|---|
| 「グローバルに活躍したい」 | 具体性ゼロ。全員が言う |
| 「日本のプレゼンスを高めたい」 | 抽象的。プレゼンスとは何か説明できないと即死 |
| 「社会に貢献したい」 | どのような社会課題に、どのようなメカニズムで貢献するのか |
| 「英語を使いたい」 | 英語は手段であり目的ではない |
| 「専門性を身につけたい」 | どの専門性で、なぜその専門性でないといけないのか |
| 「大きな仕事がしたい」 | なぜ「大きな仕事」が重要なのか。規模より中身を語れ |
| 「成長できる環境だから」 | 自己成長が目的に聞こえる。会社は成長したい学生のための機関ではない |

## 付録: 自分の志望動機の診断方法

以下の質問に**全て答えられれば**、志望動機として機能している:

**IBD**:
1. 「なぜM&Aアドバイザリーなのか？コンサルや事業会社でも同じことができるのでは？」
2. 「あなたが気になるM&Aを教えてください。そのスキーム・財務条件・戦略的意義は？」
3. 「5年後のIBDキャリアはどんな状態になっていたいですか？」

**アセマネ**:
1. 「今日一番注目している株・セクターを教えてください。理由は？」
2. 「Buffett方式のValue InvestingとGrowth Investingの違いを説明してください」
3. 「アクティブ運用とパッシブ運用の違いと、今後の展望は？」

**商社**:
1. 「受験先の商社がやった案件でひとつ、なぜその案件が戦略的に重要か説明してください」
2. 「三菱商事と三井物産の事業ポートフォリオの違いを教えてください」
3. 「商社の仕事と投資ファンドの仕事の違いは？」
