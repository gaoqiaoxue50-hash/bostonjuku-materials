## 📌 Part 1: 必須知識まとめ

### 1.1 商社選考スケジュール（vs 外銀の違い）

| 時期 | 外銀IBD | 総合商社 |
|---|---|---|
| 7〜9月 | 面接・内定（ボスキャリ前に決着） | まだ選考なし |
| 11月（ボスキャリ） | 既に内定者多数 | 1次面接（落とさないに近い） |
| 11月金曜ディナー | 木曜が本番 | 金曜ディナーが懇親会に近い |
| 翌年2〜6月 | 内定済みが多い | **日本で最終面接（本番はここ）** |

**商社就活の最重要認識**: ボスキャリで商社の1次面接を通過して「受かった」と思う学生が多いが、日本の最終面接でバッサリ落とす構造になっている。ボスキャリを通過してからが本番。

**例外**: 三菱商事は2025年からボストンで最終面接を実施（他社は追随予定なし）。

### 1.2 商社面接の質問体系（一般100選 + 商社特有60選）

#### カテゴリ① 基本質問（全業界共通・25問）

| 質問 | 対策のポイント |
|---|---|
| 自己紹介（1分・3分・英語） | 3パターン全て準備必須。英語は特に練習が必要 |
| なぜこの業界か / なぜこの会社か | 「なぜ他の業界ではないか」まで答えられること |
| この会社に入れなかったらどうするか | 「他社も受けている」と正直に言うのは構わないが、熱意が伝わる答え方で |
| 5年後・10年後のキャリア展望 | 具体的な事業領域・地域まで言えると強い |
| 強み・弱みを3つずつ | 弱みは「改善中」で終わらせない |
| 人生で一番頑張ったこと | 受験は絶対にNG。他を探す |
| ストレスをどう解消するか | 商社はストレス耐性を重視。具体的な方法を答える |
| 大事にしている価値観は | 商社マンの仕事と接続させる |
| ハードワーク高給 vs WLB | 商社を受けるなら前者と答えざるを得ない |

#### カテゴリ② 商社マン素質確認（商社特有・10問）

これが外銀との最大の違い。「人間性・社交性」が採用基準の中心にある。

| 質問 | 背景・意図 |
|---|---|
| ゴルフ・スポーツはしますか？ | 接待で使う。できるに越したことはない |
| お酒は好きですか？ | 商社の付き合いは飲み会が多い |
| 英語以外の言語スキルは？ | 中国語・スペイン語・フランス語・アラビア語があると高評価 |
| 自分は人に好かれるタイプだと思うか？ | 正直に答える。根拠（エピソード）も用意 |
| 趣味・休日の過ごし方 | 「勉強」「Netflix」は弱い。体験の豊かさをアピール |

#### カテゴリ③ 志望動機深掘り（13問・商社の核心）

| 質問 | 正しい答え方の方向性 |
|---|---|
| この会社に入れなかったら？ | 正直に他社も受けていると言う。「御社が第一志望」と言い張るのは逆効果の場合あり |
| 競合他社の方があなたのやりたいことができるのでは？ | 「なぜ三菱商事でないと出来ないのか」を論理的に返せることが必須 |
| そもそもなぜ日本で就職する必要がある？現地就職では？ | 海外大から日本就職を選ぶ理由を明確に |
| 事業経営をやりたいなら起業した方がいいのでは？ | 商社の「投資×スケール」の優位性を語れること |
| 日本語ができる中国人・韓国人と比べてなぜ採る？ | 自分だけの強みを語る。ありふれた答えは即アウト |

#### カテゴリ④ 三井物産 自分史（6問・物産専用）

三井物産は**幼少期から高校まで**を掘り下げる独自スタイルで知られる。

| 質問 | 対策 |
|---|---|
| 小学校・中学校では何をしていた？挫折は？ | 物産は原体験を重視する。幼少期のエピソードを棚卸しする |
| あなたの成功の定義は？ | 「売上」「数字」だけでなく人間的な成長や関係も含める |
| 今の原動力が形成されたきっかけは？ | 物産は「なぜそういう人間になったか」の因果関係を聞く |
| なぜ日本に戻りたいのか？ | 海外大生への定番質問。明確な理由が必要 |

#### カテゴリ⑤ ビジネス系質問（8問）

| 質問 | 準備の仕方 |
|---|---|
| 気になる会社の案件は？ | 受験先商社が関わった案件を1つ深掘りして説明できること |
| 三菱商事の新規事業を考えてみて（ケース） | 「なぜ商社でないとできないか」を軸に設計する |
| WTI原油価格は今いくらか / 今後は？ | 実際に調べて把握しておく（エネルギー商社特有） |
| 総合商社とPEファンドの違いは？ | 所有期間・関与の深さ・事業運営能力の違いを説明できること |

#### カテゴリ⑥ シチュエーション系（全業界・10問）

| 質問 | ポイント |
|---|---|
| 複数の上司から同時に仕事を振られた。どう優先順位をつける？ | 緊急度×重要度マトリクスで整理 + 報告・連絡・相談を忘れない |
| 上司がコンプライアンス上グレーなことをしている。どうする？ | 「まず上司に直接確認する」「それでも改善しないならコンプライアンス部門へ」 |
| 顧客に合致しない可能性のある案件。勧めますか？ | 「顧客のためになることが長期的な信頼になる」「商社の長期関係志向」を軸に |
| リスクを取った経験は？ | 具体的な数字・結果・自分の決断プロセスまで説明 |

### 1.3 商社のNG志望動機（完全リスト）

以下はどれも面接官に「それはうちじゃなくてもいいよね」と即座に返される危険な志望動機。

❌ 「途上国を救いたい」（でも途上国に行ったことがない）  
❌ 「再生エネルギー/風力発電/海水淡水化をやりたい」（どの商社でも同じ回答が出回っている）  
❌ 「繊維事業をやりたい」（伊藤忠で言うと激詰めされる。差別化になっていない）  
❌ 「エネルギーの安定供給に貢献したい」（ENEOSやINPEXに行けと言われる）  
❌ 「事業投資がしたい」（何の事業か？どの地域か？抽象的すぎる）  
❌ 「グローバルに働きたい」（外資系の方がグローバルでは？）  
❌ 「日本のプレゼンスを上げたい」（プレゼンスとは何ですか？）  
❌ 「日本と海外の懸け橋になりたい」（外交官になれと言われる）  
❌ 「AI事業をやりたい」（テック専業企業の方が向いている）  
❌ 「幅広い事業に携わりたい」（何がしたいのかが分からない）  
❌ 「多様なキャリアが積める」（自分のキャリアへの都合だけ）  
❌ 「スケールの大きい仕事をしたい」（具体性なし）  
❌ 「若いうちから責任ある仕事がしたい」（全ての会社がこれを言える）  
❌ 「社会に貢献したい」（どう貢献するのか？）  
❌ 「人脈が作れそう」（選社理由になっていない）

**正攻法**: 「なぜ商社というビジネスモデルで、この地域・事業・課題に取り組む必要があるのか」を具体的なビジネス構造から語ること。「なぜENEOSではなく商社か」「なぜODAではなく商社か」「なぜ起業ではなく商社か」という突っ込みに論理的に返せること。

### 1.4 Big4 FAS（財務アドバイザリー）

#### IBD vs FAS の違い

| 比較軸 | IBD（投資銀行） | FAS（Big4） |
|---|---|---|
| 主な業務 | M&A取引の主幹（資金調達含む） | DD・バリュエーション・PMI支援 |
| 関与の範囲 | 取引全体を主幹 | 特定側面（会計・財務）に特化 |
| 給与水準 | 高い（アナリストで800〜1000万） | やや低い（アナリストで600〜800万） |
| WLB | 最悪（週100時間超） | まだ悪い（週60〜80時間） |
| PEへの転職 | 外銀IBD >> 日系IBD ≧ FAS | 可能だが外銀ほど有利でない |

#### 各社の選考特性

- **Deloitte Tohmatsu Financial Advisory（DTFA）**: M&Aグループは自社の過去案件からGDが出題される傾向あり。山を張れる。複数ラウンド（夏秋冬春）ある
- **KPMG**: ケース面接がフリーディスカッション形式。普段からの思考力が問われる
- **PwC**: 戦略コンサル色が強い。ケース対策が重要
- **EY**: M&Aアドバイザリーに強み。実務的な財務知識を問う

### 1.5 ガクチカ・自己PR戦略

#### 「チームストーリー vs 個人ストーリー」問題（商社志望の最大の悩み）

商社が求めるのは「チームで大きな成果を出せるリーダー」。どちらか一方ではなく、**両方の要素が含まれること**が理想。

**理想の構造:**
```
チームで大きな目標に取り組んだ（チームワーク）
  ↓
自分がリーダーシップを発揮して状況を変えた（個人の能動性）
  ↓
具体的な成果が出た（定量化）
  ↓
そこから得た教訓が今の自分の行動原則になっている（成長）
```

#### 商社が嫌うガクチカの典型例

❌ 日本語サークル・JSA（日本学生団体）— 海外大生の定番すぎて差別化できない  
❌ 英語学習 — 全員やっている前提  
❌ 留学先での旅行 — 遊んでいるだけと見られる  
❌ 「授業を頑張りました」 — ビジネスでの貢献と繋がらない  
❌ 「ボランティアで社会貢献しました」 — 商社での収益創出とのギャップが大きい

**商社が評価するガクチカ:**
- 実際にビジネスに近い活動（起業・ビジネスコンテスト・インターン）
- 数字で成果が示せるもの（売上○○円達成・参加者○○人獲得等）
- 海外でのリーダーシップ（現地でチームを率いた経験）
