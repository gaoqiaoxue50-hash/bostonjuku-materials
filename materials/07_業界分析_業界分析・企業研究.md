## 📌 Part 1: 必須知識まとめ

### 1.1 エリートの致命的な弱点：高値掴みと損切りできないキャリア

**東大生死神理論:**

東大生がこぞって就職を希望する業界は、その時点で業績・成長のピークに達していることが多い。

| 年代 | 東大生の就職人気業界 | その後 |
|---|---|---|
| 1940年代 | セメント（戦後復興特需） | 特需終了で衰退 |
| 1950年代 | 官僚（計画経済の中枢） | 天下り厳格化で地位低下 |
| 1960年代 | 鉄鋼（インフラ特需） | 中国・インドにコモディティ化で完敗 |
| 1970年代 | 総合商社（資源バブル） | 商社冬の時代（その後復活） |
| 1980年代 | 銀行 | バブル崩壊・不良債権地獄 |
| 1990年代 | メーカー（安定神話） | 家電・テレビ・半導体で韓国・中国に完敗 |
| 2000年代 | 投資銀行 | リーマンショック |
| 2010年代 | 戦略コンサル | 生成AI登場で一部業務が代替 |
| 2020年代 | MAG7（テック） | 生成AIがコーディングを代替 |

**エリートが高値掴みをする理由:**
- 常に他人との比較・競争・上昇志向を強いられる
- 「みんなが行く業界」=「正解」と判断してしまう
- 焦りが高値掴みを誘発する

**エリートが損切りできない理由:**
- これまでの受験勉強・資格・就活への投資が大きすぎる（サンクコスト）
- 周囲の目線が気になる

**教訓**: 2026年現在、自分にとっての本質的な価値（人に好かれる力・対人スキル・AIを使いこなす力）を磨くことが重要。「流行りの業界」に飛びつくことの危険性を常に意識する。

### 1.2 業界別リサーチ手法

#### 投資銀行IBDを受ける場合

**「気になるM&A」の正しい選び方:**

| 情報源 | 特徴 | 優先度 |
|---|---|---|
| 受験先銀行のプレスリリース | 担当案件が分かる。志望動機と連動させやすい | ★★★★★ |
| EDINET（公開買付届出書） | M&Aの全詳細（財務条件・スキーム・IRR等）が記載 | ★★★★★ |
| MARR（M&A専門誌） | 業界動向・過去案件のデータベース | ★★★★ |
| 各社IR・決算説明会資料 | 企業の戦略・財務状況を把握 | ★★★ |
| 日経新聞 | 最後の手段（面接官も同じ記事を読んでいる） | ★ |

**選んではいけない案件（NG）**: USスチール・セブンアンドアイ・東芝など超大型案件  
→ 差別化できない・面接官も詳しく突っ込まれる・「日経読んでいるだけ」と判断

**理想の案件**: 受験先が担当した中型・ニッチ案件。IBD特訓シリーズ（西友3826億円等）のような具体的な分析が示せると強い。

#### 総合商社を受ける場合

1. 受験先商社の有価証券報告書：セグメント別収益から「どの事業で稼いでいるか」を把握
2. 各商社の社員インタビュー・ブログ：現場の仕事感を得る
3. 競合他社の案件を「受験先の商社」に対して語らない（三菱商事面接で三井物産の案件を語るのはNG）

#### コンサルティングを受ける場合

- McKinsey Insights / BCG Perspectives / Bain Brief（各社の公式レポート・無料）
- Diamond Harvard Business Review：経営課題の文脈把握
- 業界専門メディア（製造・金融・ヘルスケア等の専門誌）

### 1.3 業界別 内定率・英語力の実態

**ボスキャリの業界別内定率（参考値）:**
- 外資系戦略コンサル: 最も低い（定員1〜3名・超難関）
- 外銀IBD: 低い（定員少ない・テクニカル要求高い）
- 総合商社: 中程度（ボスキャリで落とさない。日本の最終が本番）
- 日系大手（不動産・損保等）: 比較的高い

**業界別 求められる英語力:**
- 外銀（外資系）: ほぼネイティブ必須（S&T・オペレーションズ系）、IBDは会話レベルで可
- 総合商社: TOEICより会話力。駐在前提なので実用的な英語
- 外資コンサル: 英語でのケース対応が必要
- 日系大手: TOEICスコアが基準になるが、実際の英語力より「やる気」を見る傾向
