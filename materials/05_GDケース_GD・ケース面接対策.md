## 📌 Part 1: 必須知識まとめ

### 1.1 ボスキャリでのGD・ケースの位置づけ

| 業界 | GD・ケースの有無 |
|---|---|
| 外銀IBD・外銀S&T | **GDなし**。1次から個人面接 |
| 外資コンサル（MBB等） | **ケース面接必須**。フェルミ推定・ビジネスケース |
| 総合商社 | 企業によりGDあり（グループ面接形式が多い） |
| Big4 FAS | DTFA: 案件ベースのGD / KPMG: フリーディスカッション型 |
| 外資消費財・FMCG | グループ選考が多い |

---

### 1.2 GD攻略の原則

**落ちる2大パターン:**
1. 話しすぎる（独占） — 自分の頭の良さを証明しようとして他人の発言を奪う
2. 話さなすぎる（存在感ゼロ） — グループの流れに乗るだけで終わる

**最も評価される役割**: 「無理にリーダーを取りに行く」のではなく「論理的で的確な貢献者」。

**時間管理の基本（30分GDの場合）:**

| 時間 | 何をするか |
|---|---|
| 0〜5分 | 問題の確認・定義・方向性の合意 |
| 5〜20分 | アイデア出し・論点整理 |
| 20〜28分 | 結論まとめ・発表準備 |
| 28〜30分 | 発表 |

---

### 1.3 ケース面接の5ステップ

外資コンサルで問われる思考プロセス:

1. **問題の明確化** — 「○○という理解でよいですか？」と確認してから答え始める
2. **仮説思考** — いきなり情報収集しない。「この問題のドライバーは何か？」を先に考える
3. **MECE分解** — 3C・4P・バリューチェーン等のフレームワークで漏れなく分解
4. **定量化** — 「大きい・小さい」ではなく「○億円・○%」で語る
5. **結論ファースト** — 結論から話し、根拠を後から説明する

**フェルミ推定の考え方の基本:**
- 分からない数値は「分けて推定する」
- 例: 「日本のコンビニの市場規模は？」
  → 日本の人口1.2億人 × 1人あたり週2回利用 × 1回あたり600円 × 52週 ≈ 7.5兆円（実際は約11兆円）
