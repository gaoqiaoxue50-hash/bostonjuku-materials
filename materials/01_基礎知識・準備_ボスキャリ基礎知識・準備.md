## 📌 Part 1: 必須知識まとめ

### 1.1 ボスキャリの全体像：「11月のイベント」は大きな誤解

ボスキャリ（ボストンキャリアフォーラム）は毎年11月にボストンで開催されるが、**実質的な選考は7〜9月に始まっている。**

| 時期 | 何が起きているか |
|---|---|
| 4月 | 国内サマーインターンのES締切（ボスキャリの前哨戦。海外大生も参加必須） |
| 6月 | CFNサイトに求人掲載開始。外銀セミナー開始 |
| 7月 | 野村・三井不動産など大手の書類選考開始 |
| 8月 | 外銀の1次面接（この時点でボスキャリは本格化） |
| 9月 | 外銀の2〜3次面接。優秀層はここで内定が出てボストン渡航を「禁止」される |
| 10月 | ディナー招待・最終面接招待 |
| 11月 | 木曜ディナー（外銀）→ 金曜ディナー（商社）→ 土曜終了。土曜中に内定が出なければ脈なし |
| 翌年2〜6月 | 商社・一部外銀の最終面接が東京で行われる |

**失敗パターン**: 9〜10月に準備を始める → 外資はほぼ終わっている  
**成功パターン**: 遅くとも3月に準備開始。ガクチカが充実しているなら5〜6月でも外資に間に合う可能性あり

**最も重要な締切**: 「一次締切」に応募した人が内定率が最も高い。ウォークイン（当日飛び込み）は最も低い。

### 1.2 参加資格と採用優先度

**採用される確率の序列（2025年現在）:**

> 海外大学部生 ≧ 海外大学院生 >> 交換留学生（在籍中） >>>>> 私費留学生 >> 語学留学・ワーホリ

近年は**ボスキャリが事実上の「海外大専用フォーラム」化**しており、交換留学生すら応募不可の企業が増加（外資戦略コンサル・三菱地所等）。

| 区分 | 現実 |
|---|---|
| 海外大学部3年生 | 主な採用対象。外銀・商社ともここが主戦場 |
| 海外大学部4年生 | フルタイム求人のみ。採用枠が縮小傾向 |
| 交換留学生（在籍中） | 留学中に応募必須。**一度帰国するとボスキャリ参加資格を失う** |
| Coop・コミカレ | 外資・商社は困難。日系大手は対策次第で可能 |
| MBA生・社会人大学院 | 転職枠・中途採用枠での応募となる |

**交換留学生が失敗しやすい7つの理由（要注意）:**
1. 留学開始後にボスキャリ準備を始める（遅すぎる）
2. 「下見」のつもりで参加（次回はないと知らない）
3. 多忙を理由に不参加
4. 海外大生に比べて英語・グローバル経験で劣ると見られがち
5. 国内選考インターンへのアクセスが物理的に難しい
6. ガクチカが「英語学習」「授業」しかない
7. 帰国後の夏採用は席数が少なく精神的プレッシャーが増す

### 1.3 何社受けるべきか：20〜30社戦略

初ボスキャリ（3年生）の推奨は**20〜30社**。「多すぎる」と感じるかもしれないが、200社中の10〜15%に過ぎない。

**4つの分類で受ける会社を組む:**

| タイプ | 目安 | 選び方のポイント |
|---|---|---|
| **練習用** | 5社 | 一番早い時期に面接してくれる会社。落ちてOK。面接感覚を掴む |
| **内定確保用** | 5〜10社 | ほぼ確実に内定が取れる会社。自信の土台を作る |
| **本命候補** | 5社 | 入社してもいい会社。ここで内定を取れれば十分 |
| **挑戦用** | 5社 | 本命中の本命。倍率が高くてもチャレンジする |

**「大手病」に注意**: 最初から難関企業だけ受けて全滅するのが典型的な失敗パターン。RPGゲームの要領でレベルを上げていく。

**ディナーの読み方:**
- **木曜（外銀）+ 金曜（商社）両日にディナー招待 = 内定に最も近いシグナル**
- 土曜ディナーはほぼ内定者懇親会（内定出し完了後）
- 内定が出やすい企業のディナーに行こうとして難関企業のディナーを断るのは悪手。チャレンジした方が結果的に内定を取る事例が多い

### 1.4 就活の本質：大学受験とはルールが違うゲーム

**大学受験**: ペーパー試験の点数が全て。採点官のご機嫌を伺う必要はない。

**就活の面接は2段階:**

| フェーズ | 性質 | 何が問われるか |
|---|---|---|
| 1〜2次面接 | **ネガティブスクリーニング** | 「この人を落とす理由はないか」を探す。論理的に話せるか、最低限の社会性があるか |
| 3次〜最終面接 | **ポジティブスクリーニング** | 「この人を採用したい理由は何か」を探す。部下にしたい人、客先に出せる人、何となく好きな人 |

**最終面接で成否を分けるのは:**
- 頭の良さ × ではない
- GPA × ではない
- 大学ランキング × ではない
- 英語力 × ではない
- **目の前の面接官に自分を好きになってもらうスキル** ○

> 「自前の頭の良さは、テスト点数のためではなく、面接官に気に入ってもらうことに全力で使うべきだ」

これを認識していない就活生は、どんな素晴らしい大学出身でも落とされる。就活はビジネスと同じ「相手にどう気に入られるか」のゲーム。

### 1.5 内定率を上げる10の準備チェックリスト

以下の10項目を確認する。当てはまる項目が多いほど内定から遠ざかっている。

| # | 失敗パターン | 対策 |
|---|---|---|
| 1 | 9〜11月に慌てて準備を始める | **少なくとも半年前（理想は1年前）から計画的に** |
| 2 | 「11月のイベント」だと思っている | ボスキャリ選考は7月から始まる。主要対象は3年生 |
| 3 | 留学経験・名門大学に過信している | 留学生同士の競争。留学以外のアピールを早期に作る |
| 4 | 面接を自己表現の場と誤解している | **面接は「営業活動」。企業が欲しい人材を演じる技術** |
| 5 | ガクチカが他の就活生と被っている | 日本語サークル・JSA・ボランティアは差別化できない |
| 6 | レジュメ（CFNレジュメ）が雑 | レジュメ = 自分の商品。美しさと読み手が知りたい情報の2点が全て |
| 7 | 「自分流」の就活に固執する | 大学受験では独自手法が通じた。就活はルールが違う |
| 8 | 大手病（難関企業だけ受ける） | 難易度の低い企業から段階的にステップアップ |
| 9 | コミュニケーション能力が低い | ディナー・面接での「明るさ」「相手の意図を察知する力」が命取り |
| 10 | 自信がない・謙遜が過ぎる | バンカー・コンサルは「人前で堂々と話せる自信」が必要条件 |

### 1.6 CFNレジュメ（ボスキャリ専用履歴書）の作り方

ボスキャリはCFN（Career Forum Network）専用のレジュメを提出する。日本のESとは異なる形式。

**学歴詳細（3000字）の2大要素:**

**① 美しさ（最重要）:**
- 採用担当者が1枚を見る時間は2〜3分
- 「美しいか否か」が最初の2秒で決まる
- 美しいレジュメ → 自己PRも読みたくなる → 面接に呼びたくなる
- レジュメ = 自分の分身・商品という意識を持つ

**② 読み手が知りたい情報:**
- 数字で定量化（GPA・奨学金・売上・順位）
- 箇条書き・体言止めが基本
- 漢字を多用（日本語として読みやすく）
- 理系は使えるプログラミング言語を必ず記載
- 「コミュニケーション力」「課題解決能力」などのありきたりな表現はNG

**自己PR（6000字）:**
- 学歴詳細より自己PRの方が圧倒的に重要
- 日本のESに相当。面接の話題の9割はここから来る
- 一番時間をかけて作り込む

### 2025年ボスキャリから学ぶ最新動向

- **外資系は採用数が年々減少**: 参加企業数・内定数ともに縮小傾向
- **「人好かれ力」が最重要**: 学歴・英語力・頭の良さよりも、人に好かれる力が内定を左右
- **ガクチカが全て**: 冬〜夏にかけて積み上げてきた学生が結果を出している
- **女性有利の傾向が継続**: コミュニケーション能力の高さが評価されやすい
- **CFNサイト以外での募集が増加**: ボスキャリに来なくてもオファーが出るケースが増えている

### 【スケジュール】

・今年も外資系を中心に書類選考と1次面接は”**7月”**から始まっていた
・11月に面接を受けるのではないのですか？スケジュールが分からず焦っていますという問い合わせが9月頃から増加（マス層）
・9月の留学が始まってから、コミカレからUC等への編入が終わってから、就活準備を行った就活生は準備が間に合わない傾向（コミカレからUCへの編入生あるいは交換留学生で、外資系などに内定している学生は、9月のずっと前から戦略的にレジュメ磨きを行っていた）
・外資系の優秀層は9-10月のボスキャリ前の段階で内定を受けて、ボストン現地入りを禁止される（他社を受けるのを強制的にシャットアウト）
・ボスキャリの半年～1年前から準備を行っていた就活生は順当に内定を得ていた**（準備が早いほど内定率が上がる）**
・非金融専攻の学生でも、9月頃からの短期間の金融勉強キャッチアップで、外資金融に内定した事例を観測
・三菱商事はボストンで最終面接を実施（今年から）
・総合商社面接は10月からとゆっくり目
・日系IBD/GMは9月
